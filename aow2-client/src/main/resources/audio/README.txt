Art of War 2: Online — Audio Assets Directory

This directory should contain the following audio files for the game client:
- ui_click.wav       — UI button click sound effect
- ui_hover.wav       — UI button hover sound effect
- build_complete.wav — Building construction completed notification
- unit_produced.wav  — Unit production completed notification
- research_done.wav — Research completed notification
- error.wav          — General error/warning sound
- explosion.wav      — Generic explosion sound effect
- gunshot.wav        — Bullet/small arms fire
- rocket_fire.wav    — Rocket launcher fire
- artillery.wav      — Artillery bombardment
- flame.wav          — Flame thrower sound
- mine_place.wav     — Mine placement sound
- mine_explode.wav   — Mine detonation
- victory.wav        — Match victory fanfare
- defeat.wav         — Match defeat sound
- ambient_war.wav    — Background ambient track (loopable)

NOTE: These are placeholder filenames. Original game audio assets
should be extracted from the Art of War 2 JAR resources or recorded
from gameplay. Until then, the AudioManager will silently skip
missing files rather than crashing.
